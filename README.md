# Sistema de cadastro
Projeto para criação de um sistema de cadastro para universidade. O sistema comporta:

- Cadastro de Pessoa Física ​
- Cadastro de Pessoa Jurídica ​
- Cadastro de Professores ​
- Cadastro de Fornecedores ​
- Cadastro de Alunos
